# Credit Risk Prediction App

## Overview
This project predicts whether a customer is a Good Risk or Bad Risk using Machine Learning.

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Streamlit
- Joblib

## Machine Learning Models
- Decision Tree
- Random Forest
- Extra Trees
- XGBoost

## Model Accuracy
- Decision Tree: 67.5%
- Random Forest: 73.5%
- Extra Trees: 71.5%
- XGBoost: 73.5%

## Features
- Interactive Streamlit UI
- Real-time prediction
- Risk confidence score
- Applicant summary
- Risk analysis logic
- Model comparison dashboard
